Formulario de registro con Validaciones HTML

  • Uso de fieldset y legend para mejorar la accesibilidad y organización del formulario.

Contenido Multimedia y Responsive

Implementación de contenido multimedia optimizado para la web, incluyendo imágenes responsivas, uso de picture para art direction y carga eficiente de video y audio. Además, se aplicaron buenas prácticas de optimización web relacionadas con peso de archivos, formatos adecuados y mejora del rendimiento, impactando métricas como tiempo de carga inicial y Largest Contentful Paint (LCP).

Lo que aprendí:

  • Uso de srcset y sizes para adaptar imágenes según el dispositivo
  • implementación de múltiples formatos de video (MP4, WebM) para compatibilidad, y optimización de carga mediante atributos como loading y preload.
  • HTML5
  • CSS3
